# 实验 4：多 Agent 客服 vs 单 Agent 客服

## 实验目标

## 实验内容

同一客服任务，用单 Agent 和多 Agent 分别实现，对比完成率、成本、速度。

## 运行方式

```bash
pip install openai rich
python compare.py
```
