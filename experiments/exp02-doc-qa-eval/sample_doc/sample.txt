# 实验 2 示例文档

这是一篇关于人工智能发展历程的综述文章，用于本实验的 RAG 问答测试。

## 1. 人工智能的起源

人工智能（AI）的概念最早可以追溯到 1956 年的达特茅斯会议。在那次会议上，约翰·麦卡锡（John McCarthy）首次提出了"人工智能"这个术语。早期的 AI 研究主要集中在符号推理和问题求解上，代表性的工作包括 Newell 和 Simon 的 Logic Theorist 程序，它能够证明数学定理。

## 2. 机器学习的兴起

1990 年代，AI 研究经历了从符号主义到连接主义的转变。统计机器学习方法开始崭露头角，代表性的算法包括支持向量机（SVM）、随机森林和 Boosting 方法。这些算法在图像识别、文本分类等任务上取得了显著的成功。这一时期的标志性事件包括 1997 年 IBM 的深蓝（Deep Blue）击败国际象棋世界冠军卡斯帕罗夫。

## 3. 深度学习革命

2012 年，AlexNet 在 ImageNet 图像识别竞赛中取得了突破性的成绩，标志着深度学习时代的到来。深度神经网络通过多层非线性变换，能够自动学习数据的层次化特征。关键的创新包括卷积神经网络（CNN）用于图像处理、循环神经网络（RNN）和长短期记忆网络（LSTM）用于序列数据。

2016 年，DeepMind 的 AlphaGo 击败围棋世界冠军李世石，这被认为是 AI 发展的又一个里程碑。AlphaGo 结合了深度学习和强化学习，通过自我对弈不断提升棋力。

## 4. 大语言模型时代

2018 年，Google 发布了 BERT 模型，在 11 项 NLP 任务上取得了最先进的结果。随后，OpenAI 在 2020 年发布了 GPT-3，展示了大规模语言模型的惊人能力。GPT-3 拥有 1750 亿参数，能够完成翻译、写作、代码生成等多种任务。

2022 年底，ChatGPT 的发布让大语言模型进入了公众视野。2023-2024 年，各大公司纷纷推出自己的大模型：Google 的 Gemini、Anthropic 的 Claude、Meta 的 LLaMA 系列。这些模型在推理、编程、数学等能力上持续提升。

## 5. Agent 系统的出现

2024-2025 年，AI 研究的焦点从单纯的模型能力转向了 Agent 系统。Agent 系统让 LLM 能够调用外部工具、访问知识库、执行多步骤任务。代表性的框架包括 LangChain、AutoGen、CrewAI 等。Agent 系统也被称为"LLM 的操作系统"，它让模型能够与现实世界交互。

ReAct（Reasoning + Acting）范式是 Agent 系统的核心，它通过"思考-行动-观察"的循环让模型能够自主解决问题。2025 年，MCP（Model Context Protocol）的推出进一步标准化了 Agent 与外部工具的交互方式。

## 6. 未来展望

AI Agent 领域的主要挑战包括：可靠性的提升（减少幻觉）、多 Agent 协作的效率、安全性和可控性、成本和效率的优化。未来的研究方向包括 Agent 的自主学习与进化、多模态 Agent、以及 Agent 在垂直行业的深度应用。

## 数据统计

根据 2025 年的行业报告：
- 全球 AI 市场规模达到 5000 亿美元
- Agent 相关产品的年增长率超过 150%
- 超过 60% 的企业已经在生产环境中使用某种形式的 AI Agent
- RAG（检索增强生成）是最广泛采用的 Agent 技术
